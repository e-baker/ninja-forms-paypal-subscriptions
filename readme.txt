== Description ==
The Ninja Forms PayPal Subscriptions Extension allows users to accept payments through their forms using the PayPal Subscriptions system.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the `ninja-forms-paypal-subscriptions` directory to your `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Visit the 'Forms' menu item in your admin sidebar
4. When you create a form, you will have now have PayPal options on the form settings page.
